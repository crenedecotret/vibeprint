# VibePrint Studio Icon

## Current Setup
The application loads `vibeprint-studio.png` as the window icon.

## Recommended Free Icons

### Option 1: UXWing Printer Icon (Easiest)
- **License**: Free for commercial use, no attribution required
- **URL**: https://uxwing.com/printer-icon/
- **Steps**:
  1. Download the 512x512 PNG
  2. Rename to `vibeprint-studio.png`
  3. Replace the file in this folder

### Option 2: IconScout Color Printer (Customizable)
- **License**: Free with attribution or premium
- **URL**: https://iconscout.com/icons/color-printer
- Can customize colors to rainbow gradient online before downloading

### Option 3: Phosphor Icons
- **License**: MIT License (free, open source)
- **URL**: https://phosphoricons.com/
- Search for "printer" icon
- Choose duotone or fill style in rainbow colors

### Option 4: Flaticon
- **URL**: https://www.flaticon.com/free-icons/printer
- Search "color printer" or "rainbow printer"
- 31,800+ printer icons available
- Check license on individual icons (most free with attribution)

## Technical Details
- **Recommended size**: 256x256 or 512x512 pixels
- **Format**: PNG with transparency
- **Fallback**: If no icon file exists, the app will use the default system icon

## Creating a Custom Icon

For a truly unique VibePrint Studio icon, consider:
- A stylized printer with a rainbow gradient paper output
- CMYK color dots forming a print head
- A paper sheet with colorful ink splashes
- Simple "VP" monogram in printing colors

Tools for creating custom icons:
- **Figma** (free): https://figma.com
- **Inkscape** (free, open source): For SVG icons
- **GIMP** (free, open source): For PNG icons
